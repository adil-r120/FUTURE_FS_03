# Brew & Bloom

Welcome to the Brew & Bloom project!

## Project Info

This is the repository for the Brew & Bloom local business website.

## Getting Started

To get started with development:

1.  **Clone the repository:**
    ```sh
    git clone <YOUR_GIT_URL>
    cd <YOUR_PROJECT_NAME>
    ```

2.  **Install dependencies:**
    ```sh
    npm install
    ```

3.  **Start the development server:**
    ```sh
    npm run dev
    ```

## Technologies Used

- Vite
- TypeScript
- React
- shadcn-ui
- Tailwind CSS

You can deploy this project using your preferred hosting provider (e.g., Vercel, Netlify).
