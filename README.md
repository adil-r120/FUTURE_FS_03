## Getting Started

To get started with development:

1.  **Clone the repository:**
    ```sh
    git clone <https://adil-r120.github.io/FUTURE_FS_03/>
    cd <Local Business Website>
    ```

2.  **Install dependencies:**
    ```sh
    npm install
    ```

3.  **Start the development server:**
    ```sh
    npm run dev
    ```
